Alert

A simple success alert.
A simple danger alert.
A simple warning alert.
A simple info alert.

Add the alert class to element. Use the alert-success class for a success alert, alert-danger tag for a danger alert, alert-warning tag for a warning alert and alert-info class for an info alert. 


    <div class="alert alert-success">...</div>
    <div class="alert alert-danger">...</div>
    <div class="alert alert-warning">...</div>
    <div class="alert alert-info">...</div>
