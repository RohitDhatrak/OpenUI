Avatar

Standard Avatars

Add the avatar class to element. There are 4 additional sizes available, including avatar-xl (4em), avatar-lg (3em), avatar-sm (1.5em), and avatar-xs (1em). By default, the avatar size is 2em. 


    <figure class="avatar avatar-xl">
        <img src="..." alt="..."/>
    </figure>

    <figure class="avatar">
        <img src="..." alt="..."/>
    </figure>

Avatars with status

Avatars support presence indicators. You can add an element with the avatar-presence class, and add online or dnd class for different status colors. The default is gray which means offline. 


    <figure class="avatar avatar-xl">
        <img src="..." alt="..."/>
        <div class="avatar-presence dnd">
    </figure>

    <figure class="avatar-lg">
        <img src="..." alt="..."/>
        <div class="avatar-presence">
    </figure>
