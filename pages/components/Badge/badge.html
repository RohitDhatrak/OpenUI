Badge

Avatars with Badge

Add a div in the Avatar component wiht a class badge-count. Check out the Avatar component for instrucions on how to use it. 


    <figure class="avatar avatar-xl badge">
        <img src="..." alt=""/>
        <div class="badge-count">8</div>
    </figure>
    
    <figure class="avatar badge">
        <img src="..." alt=""/>
        <div class="badge-count">8</div>
    </figure>

Icons with Badge

Add the badge class to a non self closing element. Add the icon in the img tag. Add a div having a class badge-count with the content that should be shown in the badge. 


    <figure class="badge">
        <img src="..." alt="" />
        <div class="badge-count">8</div>
    </figure>

    <figure class="badge">
        <img src="..." alt="" />
        <div class="badge-count">8</div>
    </figure>
